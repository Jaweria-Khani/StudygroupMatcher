# 📌 Project Title: Study Group Matcher App

## 🎯 Purpose of Prototype

To test:

* How students search and join study groups
* Whether the task flow is simple and clear
* If users understand how to create or join a group

This is a **low-fidelity (paper-based) prototype** to explore design ideas.

---

# 🎨 How to Draw It (Important for Marks)

✅ Use pencil
✅ Use rectangles for screens
✅ Use circles for profile icons
✅ Use arrows to show navigation
✅ Keep it simple (no colors needed)
✅ Label each screen clearly
