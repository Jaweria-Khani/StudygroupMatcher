Task 1: UI Mockup Design
Study Group Matcher (Web-Based Application)
1. Introduction

The Study Group Matcher is a web-based application that helps students find and join study groups for different subjects. The system allows users to search for subjects, view available study groups, and join a group to collaborate with other students.

The UI mockup for this project was designed using a free design tool such as Figma. The mockup focuses on creating a simple and user-friendly interface for students.

Main Screens in the UI Mockup
1. Login Page

Purpose:
This page allows students to log in to the system.

Main Elements:

Email input field

Password input field

Login button

Register link for new users

Explanation:
Users must enter their email and password to access the platform. This ensures that only registered students can use the system.

2. Home Page (Dashboard)

Purpose:
The home page is the main screen where users can navigate to different features.

Main Elements:

Website logo/title: Study Group Matcher

Navigation menu:

Home

Find Groups

My Groups

Profile

Search bar for subjects

Button: Find Study Group

Explanation:
This page helps users quickly find study groups for their subjects.

3. Subject Selection Page

Purpose:
Allows students to select the subject they want to study.

Main Elements:

Search bar

List of subjects such as:

Data Structures

Calculus

Database Systems

Artificial Intelligence

Select button

Explanation:
Users choose a subject to see the available study groups related to that subject.

4. Study Group List Page

Purpose:
Shows available study groups for the selected subject.

Main Elements:

Group name

Number of members

Study time

Location

Button: View Details

Example:

Group A – 3 Members – 3 PM
Group B – 5 Members – 4 PM

Explanation:
Students can browse available groups and choose the one that matches their schedule.

5. Group Details Page

Purpose:
Shows detailed information about the selected study group.

Main Elements:

Group name

Subject name

Number of members

Meeting time

Meeting location

Button: Join Group

Explanation:
Students can view the group information and decide whether they want to join.

6. My Groups / Chat Page

Purpose:
Allows students to communicate with their study group members.

Main Elements:

Group name

Member list

Chat messages

Message input box

Explanation:
Students can discuss topics, share ideas, and help each other prepare for exams.

Design Tool Used

The UI mockup was created using Figma, which is a free and popular interface design tool. It allows designers to create layouts, buttons, and screen flows easily.

Conclusion

The Study Group Matcher UI mockup provides a simple and organized interface that helps students quickly find and join study groups. The design focuses on easy navigation, clear information, and collaboration between students.
